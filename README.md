# Wizaplace PHP Code Sniffs

Wizaplace PHP code sniffs

## Installation

```bash
composer require --dev wizaplace/phpcs
```

```xml
<?xml version="1.0"?>
<ruleset>
    <rule ref="vendor/wizaplace/phpcs/BaBeuloula/ruleset.xml"/>
</ruleset>
```
